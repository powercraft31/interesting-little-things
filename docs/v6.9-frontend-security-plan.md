## 3. CSP Blocker Cleanup — Summary

All CSP blockers from the design doc are addressed by the slices above:

| Blocker (Design §6.1) | Resolved By | Slice |
|------------------------|-------------|-------|
| B1 — ECharts CDN script | Self-hosted `js/vendor/echarts.min.js` | F1 |
| B2 — Google Fonts CSS | Self-hosted fonts + `@font-face` in CSS | F2 |
| B3 — Inline `<script>` in `login.html` | Extracted to `js/login.js` | F3 |
| B4 — Inline `onclick` in `login.html` | Replaced with `addEventListener` | F3 |
| B5 — Google Fonts `preconnect` | Removed after self-hosting | F2 |

After all five blockers are resolved, the CSP baseline (`script-src 'self'; style-src 'self'; font-src 'self'`) is enforceable with no exceptions.

## 5. Slice Execution Order and Dependencies

```
F1 (Self-host ECharts)   ──── no dependency ──────────────────────────┐
F2 (Self-host fonts)     ──── no dependency ──────────────────────────┤
F3 (Extract login.js)    ──── no dependency ──────────────────────────┤
                                                                      │
                          ┌── WS-B: GET /api/auth/session ready ──┐   │
F4 (Session bootstrap)   ──── BLOCKED on WS-B ───────────────────┤   ├── G2
                                                                  │   │
                          ┌── WS-B: cookie-first middleware ──────┤   │
F5 (Bearer removal)      ──── BLOCKED on WS-B ───────────────────┘───┘
```

**Parallelism**: F1, F2, F3 can all execute concurrently and have no backend dependency. F4 and F5 require the backend session endpoint and cookie-first middleware respectively, but can run concurrently with each other once their dependencies are met.

## 6. Bounded Open Decisions — Resolution During Implementation

| Decision | Options | Constraint | Decide At |
|----------|---------|------------|-----------|
| `@font-face` location | Add to `variables.css` (simpler) or create `fonts.css` (separation) | Either acceptable; `variables.css` preferred for zero-build constraint | Slice F2 start |
| ECharts exact version | Check current CDN URL in `index.html` for version | Must pin to exact same version | Slice F1.1 |
| `login.js` IIFE structure | Wrap in `(function() { ... })();` | Consistent with other JS files in codebase (no ES modules) | Slice F3.2 |
| Font file naming | `inter-400.woff2` vs. `Inter-Regular.woff2` | Either acceptable; short lowercase preferred | Slice F2.1 |
