## 2. M3 Reverse Command Path — Full Flow

### Current State

The existing `command-dispatcher.ts` in `backend/src/dr-dispatcher/services/` handles `trade_schedules → dispatch_commands` (M2→M3 internal). But there is **no path** from Frontend PUT `/api/devices/:id/schedule` → `device_command_logs` → MQTT publish.

### Flow Diagram

```
Frontend (P2 Device Detail → "Apply to Gateway" button)
  │
  ▼
BFF PUT /api/devices/:assetId/schedule
  │  Payload: { slots: [{ startHour, endHour, mode }] }
  │  Handler: backend/src/bff/handlers/put-device-schedule.ts (exists)
  │
  ▼
INSERT INTO device_command_logs (
  gateway_id, asset_id, command_type, payload,
  status, created_at
) VALUES (
  <asset's gateway_id>, $assetId, 'schedule_update',
  $payload::JSONB, 'pending_dispatch', NOW()
)
  │
  ▼
M3 command-dispatcher (10s polling loop)
  │  Query: SELECT * FROM device_command_logs
  │         WHERE status = 'pending_dispatch'
  │         ORDER BY created_at ASC
  │         LIMIT 50
  │         FOR UPDATE SKIP LOCKED
  │
  ▼
For each pending command:
  1. UPDATE status = 'dispatching', dispatched_at = NOW()
  2. MQTT publish to: platform/ems/{gateway_id}/config/set
     Payload: { assetId, commandType, payload, commandLogId }
  3. UPDATE status = 'dispatched'
  │
  ▼
M1 MQTT ACK listener (backend/src/iot-hub/handlers/command-tracker.ts)
  │  Subscribes to: platform/ems/{gateway_id}/config/ack
  │  On ACK: UPDATE device_command_logs SET status='acknowledged', acked_at=NOW()
  │           WHERE id = $commandLogId
  │
  ▼
Timeout check (run alongside dispatcher or separate cron):
  UPDATE device_command_logs
  SET status = 'timeout'
  WHERE status = 'dispatched'
    AND dispatched_at < NOW() - INTERVAL '90 seconds'
```

```typescript
export async function runPendingCommandDispatcher(pool: Pool): Promise<void> {
  const client = await pool.connect();
  try {
    await client.query("BEGIN");

    // Step 1: Fetch pending commands (LIMIT 50 per Gemini Decision #4)
    const pendingResult = await client.query<{
      id: number;
      gateway_id: string;
      asset_id: string;
      command_type: string;
      payload: Record<string, unknown>;
    }>(`
      SELECT id, gateway_id, asset_id, command_type, payload
      FROM device_command_logs
      WHERE status = 'pending_dispatch'
      ORDER BY created_at ASC
      LIMIT 50
      FOR UPDATE SKIP LOCKED
    `);

    if (pendingResult.rows.length === 0) {
      await client.query("COMMIT");
      return;
    }

    const ids = pendingResult.rows.map((r) => r.id);

    // Step 2: Mark as dispatched
    await client.query(
      `UPDATE device_command_logs
       SET status = 'dispatched', dispatched_at = NOW()
       WHERE id = ANY($1)`,
      [ids],
    );

    // Step 3: Publish MQTT for each command
    for (const cmd of pendingResult.rows) {
      const topic = `platform/ems/${cmd.gateway_id}/config/set`;
      const message = {
        commandLogId: cmd.id,
        assetId: cmd.asset_id,
        commandType: cmd.command_type,
        payload: cmd.payload,
        timestamp: new Date().toISOString(),
      };
      await publishMqtt(topic, JSON.stringify(message));
    }

    await client.query("COMMIT");

    console.log(
      `[PendingCommandDispatcher] Dispatched ${pendingResult.rows.length} commands`,
    );
  } catch (err) {
    await client.query("ROLLBACK");
    console.error("[PendingCommandDispatcher] Error:", err);
  } finally {
    client.release();
  }
}
```

## 4. 0 vs null Rule — COALESCE Keep/Remove Matrix

### Rule

| Context | Pattern | Keep/Remove | Reason |
|---------|---------|:-----------:|--------|
| Aggregate (SUM/AVG/COUNT across rows) | `COALESCE(SUM(...), 0)` | **KEEP** | Zero-row aggregate = legitimately zero |
| Single-entity telemetry field | `parseFloat(...) \|\| 0` | **REMOVE** → `null` | Null = "no data received" |
| Hardcoded fallback value | `value: 45` | **REMOVE** → `null` | No real data source |
| Tariff fallback | `0.85 / 0.41 / 0.62` | **REMOVE** → `null` | No tariff = unknown cost |

```typescript
/** Parse numeric DB value; returns null if input is null/undefined. */
function toNum(val: unknown): number | null {
  if (val == null) return null;
  const n = parseFloat(String(val));
  return Number.isNaN(n) ? null : n;
}

function toInt(val: unknown): number | null {
  if (val == null) return null;
  const n = parseInt(String(val), 10);
  return Number.isNaN(n) ? null : n;
}
```
