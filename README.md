# SOLFACIL VPP Demo

Virtual Power Plant (VPP) demonstration system for SOLFACIL Brazil pilot project.

## Overview

This demo showcases a financial-first approach to battery energy storage management, designed specifically for SOLFACIL - Brazil's largest solar fintech company.

## Features

- **Financial Dashboard**: Focus on ROI, arbitrage opportunities, and revenue tracking
- **Real-time Monitoring**: Battery status, energy flow, and market prices
- **Batch Control**: Manage multiple battery assets simultaneously
- **Multi-language Support**: Portuguese and English interfaces
- **Responsive Design**: Works on desktop and mobile devices

## Technology Stack

- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **Charts**: Chart.js for data visualization
- **Icons**: Font Awesome
- **Styling**: Custom CSS with Material Design principles

## Project Structure

```
├── index.html          # Main application
├── app.js             # Core application logic
├── style.css          # Main styles
├── css/               # Modular CSS files
├── js/                # JavaScript modules
├── backup/            # Version backups
└── test-*.html        # Test pages
```

## Installation

1. Clone the repository
2. Serve the files using any web server
3. Open `index.html` in a web browser
